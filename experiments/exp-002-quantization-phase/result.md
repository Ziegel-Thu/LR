### Phase 1 分析

#### 1. 权重量化比激活量化鲁棒得多
- Pilot（激活量化）：4-bit mean_r = 0.720
- Phase 1（权重量化）：4-bit mean_r = 0.942–0.997（取决于层）
- 权重量化下 **4-bit 几乎无损**，这与业界 INT4 部署经验一致

#### 2. 中间层最脆弱
- L12/L17（模型中段）受量化影响最大：2-bit 时 mean_r ≈ 0.45
- L0（embedding 附近）极其鲁棒：2-bit 仍然 0.964
- L23（接近输出）介于两者之间
- 解释：中间层承担最复杂的特征变换，superposition 更紧凑，量化误差影响更大

#### 3. 稀有特征先崩溃——但效应比 pilot 弱
- 频率分层效应在权重量化 + 大模型上**复现**
- 但 rare vs common 差距比 pilot 小得多（2-bit L17: 0.260 vs 0.445，而非 0.045 vs 0.715）
- 特别注意 L0 层出现反转：最常见特征恢复率反而比 Q2-Q4 低
- 这可能因为权重量化的误差模式不同于激活量化（误差经网络传播后会被重新分配）

#### 4. 退化曲线更陡——接近 sharp transition
- 权重量化的退化曲线比激活量化更陡：
  - 6→4 bit：几乎无变化
  - 4→3 bit：开始退化
  - 3→2 bit：**剧烈崩溃**（L12 从 0.843 → 0.467）
- 这个 4→3→2 的阶梯形退化比 pilot 的 smooth 下降更接近"临界 bit-width"假说

#### 5. 局限性
- 只有一个模型（Gemma-2 2B），无法做有限尺寸标度分析
- 没有对照组（如匹配 MSE 的高斯噪声、GPTQ 等其他量化方法）
- 需要 Pythia scaling ladder 来区分 smooth crossover 和 true phase transition
