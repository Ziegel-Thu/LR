# DR-012: 跨领域不可能性定理类比搜索——寻找值得定义的新问题

## 动机

好的研究问题往往来自"把 A 领域已有的公理化不可能性定理搬到 B 领域，发现对应物尚未被研究"。例如：
- Arrow 1951（社会选择）→ Kleinberg 2002（聚类）：都是"三个合理公理不相容"
- Heisenberg 不确定性（量子力学）→ 率失真理论（信息论）：都是"精度和效率不可兼得"
- Gödel 不完备性（数学逻辑）→ Rice 定理（可计算性）：都是"足够丰富的系统无法自我判定"

我们在研究**神经网络表征分析方法的不可能性**。已知四个部分结果（Bilodeau, Han, Sutter, Kleinberg），并自己证明了一个 mini impossibility（additivity 是根源）。

**目标**：系统性地搜索其他领域的不可能性定理，看哪些在表征分析中有未被研究的对应物——这些对应物就是值得定义的新问题。

## 需要搜索的领域

### 1. 信息论

- **率失真理论 (Rate-Distortion Theory)**：在给定失真上限下，压缩率有不可突破的下界。表征分析中的对应物是什么？（$\Phi$ 是"压缩"，faithfulness 是"失真"）
- **数据处理不等式 (Data Processing Inequality)**：处理不会创造信息。表征分析方法是否也受此约束——$\Phi$ 的输出不可能比输入（模型本身）包含更多关于行为的信息？
- **信道容量定理**：有噪信道有容量上限。表征分析是否可以看作噪声信道？
- **Blackwell 充分性 (Blackwell's Theorem)**：实验 A 比实验 B "更有信息"当且仅当 B 是 A 的后处理。不同表征分析方法之间是否存在 Blackwell 序？

### 2. 统计学

- **Cramér-Rao 下界**：无偏估计量的方差有下界（Fisher 信息的逆）。表征分析方法作为"估计量"，其精度是否也有类似下界？
- **Rao-Blackwell 定理**：充分统计量的条件期望改进任何估计量。表征分析中是否存在"充分统计量"——一个所有方法都可以从中派生的最小充分表示？
- **Lehmann-Scheffé 完备性**：完备充分统计量的无偏估计是唯一最优的。这是否暗示"如果表征分析的充分统计量是完备的，则最优方法唯一"？
- **James-Stein 悖论**：在 d≥3 时，MLE 不是 admissible 的——可以通过 shrinkage 改进。表征分析方法是否也有类似的"高维下所有 naive 方法都 inadmissible"的结果？

### 3. 经济学 / 社会选择

- **Gibbard-Satterthwaite 定理**：非独裁的投票系统必然可操纵。表征分析方法是否也"可操纵"——是否总能构造模型使任何方法输出预设的解释？（这和 Bilodeau 的 adversarial 构造很相关）
- **Myerson-Satterthwaite 定理**：双边交易中效率+预算平衡+个体理性+激励相容不可同时满足。四个公理的不相容——能否映射到表征分析？
- **Sen 的不可能性 (Impossibility of a Paretian Liberal)**：最小自由+帕累托最优不相容。"方法自由度"和"全局最优性"在表征分析中的矛盾？
- **VCG 机制作为正面结果**：Arrow 之后，VCG 通过放松一个公理（不要求序数比较，只要求准线性偏好）恢复了可能性。表征分析中类似的"放松一个公理后的最优方案"是什么？

### 4. 计算复杂度

- **PCP 定理 / 近似不可能性**：某些优化问题不仅 NP-hard，连近似都 NP-hard。表征分解（如 SAE）的最优解是否也有近似不可能性？
- **通信复杂度下界**：两方持有各自的输入，要计算联合函数需要多少通信。类比：两个表征分析方法（如 probe 和 SAE）各自持有部分信息，要得到完整解释需要多少"通信"？
- **属性检测 (Property Testing)**：用亚线性查询判断函数是否满足某性质。表征分析方法能否用亚线性采样判断模型是否有某特征？

### 5. 拓扑学 / 微分几何

- **Borsuk-Ulam 定理**：连续映射 $S^n \to \mathbb{R}^n$ 必有对径点同像。降维的表征分析（d_model → k 个特征，k < d_model）是否必然"混淆"某些本质不同的表征？
- **Nash 嵌入定理**：任何黎曼流形可以等距嵌入到足够高维的欧氏空间。表征流形是否也可以——但需要多高的维度？
- **不动点定理 (Brouwer / Kakutani)**：连续映射有不动点。表征分析方法的迭代（如 SAE → 解释 → 改进 SAE）是否保证收敛？

### 6. 测量理论 / 物理

- **海森堡不确定性原理**：$\Delta x \cdot \Delta p \geq \hbar/2$。在表征分析中，是否存在两个互补的"可观测量"（如局部特征 vs 全局结构）不能同时精确测量？
- **观察者效应**：测量改变被测系统。Activation patching 就是这样——干预改变了模型的内部状态。是否有精确的量化？
- **Bell 不等式**：某些量子关联不能被局部隐变量模型解释。表征之间的某些关联是否也不能被线性方法解释？（这和 Sutter 的 trivialization 有关）

## 输出期望

对每个领域的每个定理：

1. **原始定理的精确陈述**（一句话版本）
2. **在表征分析中的候选对应物**（如果能想到）
3. **这个对应物是否已被研究？**（附文献引用，如果有）
4. **如果未被研究，值不值得做？**（评估难度、影响力、与我们现有工作的协同性）
5. **前 5 名推荐**：综合考虑新颖性、可行性、影响力，推荐最值得追的 5 个方向

特别关注：
- 与我们已有的 exp-005 发现（worst-case vs average-case gap）能产生协同的方向
- 能导向**正面结果**（不仅仅是不可能性，还有"放松公理后的最优方案"）的方向
- 有**实验可验证**的理论预测的方向
