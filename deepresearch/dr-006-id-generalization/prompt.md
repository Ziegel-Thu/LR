# DR-006: 内蕴维数与泛化因果链——实验方案设计

## 背景

多项工作发现表征的内蕴维数（Intrinsic Dimension, ID）与网络泛化性能之间存在强相关：

- Ansuini et al. (arXiv:1905.12784, NeurIPS 2019) 发现表征 ID 沿网络深度呈"hunchback"曲线（先升后降），最后一层的 ID 与测试精度强相关
- Pope et al. (arXiv:2104.08894, ICLR 2021) 测得 ImageNet 的内蕴维数仅约 25-40，远低于像素维度
- Cohen, Chung & Sompolinsky (Nature Communications 2020) 将"流形容量"作为分离性度量
- Wakhloo & Chung (PRL 2023, arXiv:2211.14961) 将容量理论扩展到相关变异性

然而，**这些都是相关性证据，不是因果关系**。一个关键的开放问题是：

> "压低表征内蕴维数"是否**因果地**导致"泛化提升"？还是两者都是某个第三因素（如正则化强度、数据结构）的结果？

要建立因果链，需要设计**干预实验**：在控制其他因素的情况下，直接操纵 ID 并观察泛化变化。

相关工作：
- 内蕴维数估计方法：MLE (Levina & Bickel 2005)、TwoNN (Facco et al. 2017, arXiv:1706.02138)、PHD (Bac & Zinovyev 2020)
- 隐式正则化文献（如 weight decay、dropout）可能通过降低 ID 来改善泛化，但机制未被分离
- Sharma & Kaplan (arXiv:2004.10802) 的 scaling law α ≈ 4/d_intrinsic 建立了 ID 与 scaling 指数的联系，但仍是经验关系

## 请设计的内容

请设计干预实验建立 ID → 泛化的因果链：

1. **干预方法**：如何"直接操纵"表征的内蕴维数？
   - 可能的手段：显式 ID 正则化损失？投影到低维子空间？架构瓶颈？数据增强？
   - 关键要求：干预要足够"外科手术式"——只改变 ID，不通过其他路径影响泛化

2. **因果识别策略**：
   - 如何排除"干预同时改变了其他泛化相关因素"的可能？
   - 是否需要 instrumental variable 或其他因果推断技术？

3. **模型和数据集选择**：
   - 需要够小以便多次训练（消融），又够复杂以便 ID 有意义变化
   - 建议具体的模型架构和数据集

4. **度量**：
   - 如何准确估计训练过程中的 ID 变化？
   - 泛化用什么指标？（test accuracy？OOD generalization？calibration？）

5. **成功标准**：
   - 什么结果建立因果链？（如：ID 降低 → 泛化提升，且控制了混杂）
   - 什么结果否定因果假说？（如：ID 降低但泛化不变或恶化）

## 实验条件

4×A100 或本地 M4 Max 32G Mac（小规模预实验）。
