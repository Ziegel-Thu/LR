# DR-007: RLHF 表征漂移——实验方案设计

## 背景

大语言模型通常经历三阶段训练：预训练 → SFT（监督微调）→ RLHF/DPO（基于人类反馈的强化学习/直接偏好优化）。RLHF 阶段的目标是对齐模型行为与人类偏好，但其对模型内部表征的影响尚不清楚。

**"表面对齐假说"**（Superficial Alignment Hypothesis, Zhou et al., arXiv:2312.00752）认为 RLHF 只改变模型的"风格"而非深层能力，暗示表征变化应集中在浅层/表面层。但这一假说缺乏定量的表征几何证据。

核心问题：
1. RLHF 引起的表征变化在哪些层最大？是否集中在后层？
2. 变化是否集中在低秩子空间？（即只有少数方向被显著改变）
3. 表征漂移与能力保留/丧失之间有什么定量关系？
4. 能否从表征漂移的几何特征预测 RLHF 是否会导致安全性退化？

相关工作：
- Representation Engineering (arXiv:2312.15997): 通过直接操纵表征方向来控制模型行为
- FRPO (arXiv:2602.08813): 安全保持的偏好优化
- Online RLHF 收敛分析 (arXiv:2410.15610): 算法层面的收敛性
- Park, Choe & Veitch (arXiv:2311.03658): 线性表征假说与因果内积——为表征漂移的方向性分析提供理论基础
- DPO (Rafailov et al., arXiv:2305.18290): 简化的偏好对齐方法，适合做受控实验
- 许多开源模型同时提供 base/SFT/RLHF 版本（如 Llama 系列、Qwen 系列、Tulu 系列），为比较分析提供了便利

## 请设计的内容

请设计实验量化 RLHF 引起的表征漂移：

1. **模型选择**：哪些模型同时提供 base / SFT / RLHF(DPO) 三个阶段的 checkpoint？列出具体选择和理由

2. **表征漂移度量**：
   - 逐层分析：用什么度量比较同一层在不同训练阶段的表征？（CKA、Procrustes、权重差的谱分析等）
   - 方向性分析：漂移是否集中在特定方向？如何用 SVD/PCA 量化？

3. **低秩结构验证**：
   - 权重差 ΔW = W_RLHF - W_SFT 的有效秩是多少？
   - 与 LoRA 的低秩假设一致吗？

4. **漂移 vs 能力**：
   - 如何量化"能力保留"？（downstream benchmark 变化）
   - 表征漂移的大小/方向与能力变化之间的相关性如何？

5. **安全预测**：
   - 能否从表征漂移的几何特征（如漂移子空间与安全相关方向的夹角）预测安全性退化？

6. **成功标准**：什么结果是有意义的发现？

## 实验条件

4×A100 或 8×A40 GPU 服务器，只做推理分析，不训练模型。
