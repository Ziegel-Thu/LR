# DR-018：各种"理解型模型"上的 Probing 现状

## 动机

很多模型声称学到了某种理解能力。我们想知道：大家在这些模型上 probe 了什么、发现了什么、还缺什么？特别是有没有人 probe 过抽象层级的概念（守恒量、因果关系、反事实），而不只是简单属性。

## 任务

### 1. 按模型分类，列出已有 probing 工作

对每种模型：
- **V-JEPA 2 / I-JEPA**：probe 了什么物理属性？有没有 probe 抽象物理概念（守恒、对称性）？
- **DreamerV3 / 其他 world model**：probe 了什么状态变量？有没有 probe 反事实？
- **OthelloGPT**：state probe 已知，有没有人 probe strategy / planning？
- **CLIP / ImageBind**：probe 了什么？有没有 probe "理解" vs "模式匹配"？
- **LLM（GPT/LLaMA）**：有没有 probe "世界模型"？因果推理？反事实？
- **Coconut / Huginn（latent reasoning）**：有没有 probe 推理步骤？
- **其他声称理解的模型**

每个给：论文、probe 了什么概念、用什么 probe 方法、核心发现。

### 2. Probe 的概念层级

大家 probe 的概念可以分层级：
- 低级：像素属性、token 属性
- 中级：物体类别、语法结构
- 高级：语义关系、因果关系
- 抽象：守恒量、对称性、反事实、规划

目前各模型上 probe 到了哪个层级？哪个层级是空白？

### 3. "理解 vs 统计相关"的区分方法

有没有方法论来区分"模型真的理解了 X"和"模型只是利用了 X 的统计相关"？
- 反事实测试？
- 组合性测试？
- 分布外泛化测试？
- 已有论文怎么做的？

### 4. "理解 vs 不理解"的对照实验

有没有人做过：训练好的 vs 随机初始化的同架构模型的 probing 对照？训练充分的 vs 不充分的？学到了 vs 没学到的？
